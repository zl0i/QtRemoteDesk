# Сервер для удаленного доступка к терминалу

Sender - сторона отправляющая видеопоток и движение своей мыши, а также принимающая события (мыши и клавиатуры) от Receiver 
Receiver - сторона принимающая видеопоток, принимающая движения мыши от Sender, а также отправляющая события (мыши и клавиатуры) для Sender

Сервер совмещает в себе http сервер и websocket сервер. По http доступно создание комнаты (POST /rooms) в ответ сообщается код комнаты и url для поключения Sender для передачи видеопотока и потока событий. И получение комнаты по коду (GET /rooms/:code), в ответ сообщается код комнаты и url для поключения к websocket серверу.

### Подключение к ws серверу
Для подключения к ws серверу Sender должен к путям полученным от http сервера добавить путь /sender
Для подключения к ws серверу Receiver должен к путям полученным от http сервера добавить путь /receiver

## Сборка и запуск

### Сборка

```bash
npm run build
```

### Запуск

Для запуска нужно указать в переменных среды

```bash
WS_HOST=http://localhost # Путь до ws сервера
WS_PORT=8080 # Порт ws сервера
HTTP_PORT=3000 # Порт http сервера
```
Запуск
```bash
node .dist/main.js
```
